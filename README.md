# IT News Crawling and Mailing Automation

일일 IT 뉴스를 자동으로 수집, 요약하여 팀원들에게 전달하는 자동화 프로젝트입니다.

## 프로젝트 개요

어제의 최신 IT 뉴스를 크롤링하여 요약하고, 하이라이트를 만들어 팀원들에게 자동으로 제공하는 시스템입니다.

### 대상 뉴스 분야
1. 로보틱스 (Robotics)
2. AI (Artificial Intelligence)
3. 그외 개발 뉴스 (General Development News)

## 주요 기능

### 1. 뉴스 소스 관리
- `site-list.txt` 파일을 통한 뉴스 사이트 목록 관리
- 분야별 최적의 뉴스 소스 큐레이션

### 2. 뉴스 크롤링 및 수집
- 등록된 사이트 목록에서 자동으로 뉴스 수집
- 페이지별 컨텐츠 파싱

### 3. 뉴스 요약
- AI를 활용한 각 아티클 자동 요약
- 각 아티클당 대표 이미지 1장 선택

### 4. 데이터베이스 관리
- 요약된 뉴스를 제목 및 사이트별로 DB 저장
- 중복 뉴스 필터링 (이미 처리된 뉴스는 제외)

### 5. 동영상 생성
- 요약 내용과 대표 이미지를 활용
- Sora를 이용한 자동 동영상 생성

### 6. 메일 전송
- 팀원들에게 자동으로 뉴스 요약 전송

### 7. 마크다운 파일 출력
- 날짜별로 요약된 뉴스를 마크다운 파일로 자동 생성
- 카테고리별로 구조화된 문서
- 이미지 포함 (이미지가 있는 경우)

## 기술 스택 (예정)

### Backend
- **Python 3.x**: 메인 프로그래밍 언어
- **BeautifulSoup4 / Selenium**: 웹 크롤링
- **OpenAI API / Claude API**: 뉴스 요약 생성
- **SQLite / PostgreSQL**: 뉴스 데이터 관리

### 외부 API
- **Sora API**: 동영상 생성
- **SMTP**: 이메일 전송

### 도구
- **Cron / Task Scheduler**: 일일 자동 실행

## 디렉토리 구조 (예정)

```
it-news-mail/
├── README.md
├── spec.md
├── site-list.txt           # 뉴스 사이트 목록
├── requirements.txt        # Python 패키지 의존성
├── config/
│   └── config.yaml        # 설정 파일
├── src/
│   ├── crawler/           # 크롤링 모듈
│   ├── summarizer/        # 요약 모듈
│   ├── video_generator/   # 동영상 생성 모듈
│   ├── mailer/            # 메일 발송 모듈
│   ├── exporter/          # 마크다운 출력 모듈
│   └── database/          # DB 관리 모듈
├── data/
│   └── news.db            # SQLite DB (또는 PostgreSQL 연결)
├── output/
│   └── markdown/          # 마크다운 파일 출력 디렉토리
├── logs/                  # 로그 파일
├── tests/                 # 테스트 코드
├── test_markdown_export.py  # 마크다운 출력 테스트 스크립트
└── main.py                # 메인 실행 파일
```

## 설치 및 사용 (추후 업데이트 예정)

```bash
# 패키지 설치
pip install -r requirements.txt

# 설정 파일 수정
# config/config.yaml 파일에서 API 키 및 설정 입력

# 실행
python main.py

# 마크다운 출력만 테스트
python test_markdown_export.py
```

## 마크다운 출력 형식

생성되는 마크다운 파일은 다음과 같은 형식입니다:

- 파일명: `news_YYYY-MM-DD.md`
- 카테고리별로 섹션 구분 (로보틱스, AI, 개발 뉴스)
- 각 뉴스 항목은 다음 정보 포함:
  - 제목
  - 대표 이미지 (이미지가 있는 경우)
  - 출처 및 날짜
  - AI 요약 내용
  - 원문 링크
